# Environment Variables

- `MONGODB_URI`, the mongodb database uri
- `PORT` the http server port. By default is `3000`

# Installation

```
git clone https://github.com/Naspi666/Inmobiliaria.git
cd proyectoInmobiliaria
npm install
npm run build
npm start
```
